# Hans Neovim Project

A customize neovim text editor using lazy.nvim for plugin manager and another plugin for user experience.
This project are suitable for programmer who want to learn about configuration a software.

---

## Features
- A beutiful status line
- Kanagawa wave colorscheme
- A auto-completion
- Tree explorer

## Requirements
- Neovim version 0.11.5 or above 0.11.x
- Lua version 5.4.6
- tree-sitter cli
- Git

## Installation

Clone the repository
```bash
git clone https://github.com/HansHazwan/nvim-setup
```
